Learn any language with a spreadsheet and a dictaphone

  1. Get yourself a dictaphone (I use a Philips voice tracer LFH0602) and record whenever you find yourself with a missing word, phrase or sentence in the language you want to learn
  2. Whenever you find time, play back the recordings and add them to your spreadsheet
  3. If you need more empty rows, put an x into cell A1.

Benefits and advantages

  • Translates automatically using Deepl and Google Translate (to compare results between the two)
  • Can be accessed from anywhere with a smartphone/tablet (Google Sheets app) or in a web browser (using a laptop or computer)
  • Never forget your missing words by using your dicatphone in the moment when you were looking for the right vocabulary.
  • Can be easily searched with the built-in search function that Google Sheets provides

How to set up the spreadsheet

  1. Make a copy of this sheet
  2. Get yourself a Deepl account to obtain your API key
  3. Edit the Google Script and add your own API key
